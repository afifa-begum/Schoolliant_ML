I developed the Student Performance Analyzer project using Python to understand how basic programming concepts work in a real-world application.
The main goal of this project is to analyze a student's academic performance by calculating marks, percentage, and grade automatically.
In this project, the program first asks the user to enter the student’s name and marks for 5 subjects. The marks are stored in a list, which makes the data easier
to manage. Then, the program calculates the total marks and percentage using mathematical operations.
After calculating the percentage, the program uses if-else conditions to assign a grade based on the student’s performance. Finally, it displays all the details 
in a neatly formatted report card.

Through this project, I learned and practiced important Python concepts such as:
-Variables
-User input
-Lists
-Functions
-Loops
-If-Else statements

This project helped me improve my logical thinking and understand how Python can be used to automate simple student record systems.
